# Cognetix Bulk Email Sender 📧

This is a Python automation project developed as part of the Cognetix Python Developer Internship.

## 🚀 Features
- Send emails to multiple recipients (simulation)
- Reads email list from a file
- Automated process with clear output
- Simple and easy to use

## 🛠️ Technologies Used
- Python
- File Handling

## ▶️ How to Run
1. Open terminal
2. Run:
   python email_sender.py

## 📂 Project Structure
- email_sender.py → Main script
- emails.txt → List of email addresses
- README.md → Documentation

## 📸 Sample Output
- Email sent messages displayed
- All emails processed successfully
